<h2>Camera App using Cordova Camera plugin for Android</h2> 
<h4>Requirements :</h4>
<ul>
  <li>Environment : JDK 1.8 (or above), Anroid SDK</li>
  <li>Debugged Tool/Emulator : Android Studio (ADV)</li>
  <li>Cordova Apache, Camera plugin installed </li>
</ul>
